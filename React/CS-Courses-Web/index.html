Dawson College Computer Science Technology

all about our courses

The Computer Science Technology Program is a three-year technical program that prepares students to work as entry-level software developers. A software developer, both individually and as a member of a team, conceives, designs and tests the logic for solving problems with a program. They write, test, debug, deploy and maintain the software applications.

Some examples of jobs Computer Science Technology Program graduates fill in industry are:

  • Full-stack Web programming
  • Mobile device programming
  • Database programming
  • Quality assurance specialist
  • Software support specialist
